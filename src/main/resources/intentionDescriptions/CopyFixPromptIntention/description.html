复制修复 Prompt

当您遇到代码问题时，使用此功能可以快速生成结构化的修复提示，方便与 AI 工具协作。

功能说明：

  • 自动获取当前文件路径和行号
  • 提取选中的代码或当前行内容
  • 自动集成剪贴板中的错误信息
  • 生成包含上下文信息的修复提示
  • 一键复制到剪贴板

使用步骤：

  1. 复制 IDE 中的错误信息（Ctrl+C）
  2. 将光标放在需要修复的代码行
  3. 按 Alt+Enter 打开意图菜单
  4. 选择"复制修复 Prompt"
  5. 将剪贴板内容粘贴到 AI 工具（如 Claude Code）

提示：

  • 使用前请先复制 IDE 中的错误信息到剪贴板
  • 错误信息可以是编译错误、警告或异常信息
  • 生成的 Prompt 将自动包含剪贴板中的错误描述
  • 支持各种编程语言的错误信息
