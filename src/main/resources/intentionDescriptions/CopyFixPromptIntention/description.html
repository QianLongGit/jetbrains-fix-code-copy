复制修复 Prompt

当您遇到代码问题时，使用此功能可以快速生成结构化的修复提示，方便与 AI 工具协作。

功能说明：

  • 自动获取当前文件路径和行号
  • 提取选中的代码或当前行内容
  • 智能获取 IDE 错误信息（无需手动复制）
  • 降级使用剪贴板中的错误信息（作为后备）
  • 生成包含上下文信息的修复提示
  • 一键复制到剪贴板

使用步骤：

  1. 将光标放在有错误或警告的代码行
  2. 按 Alt+Enter 打开意图菜单
  3. 选择"复制修复 Prompt"
  4. 将剪贴板内容粘贴到 AI 工具（如 Claude Code）

错误信息获取机制：

  • 优先级1：自动获取 IDE 中的错误描述和警告信息
  • 优先级2：如果无法获取 IDE 错误信息，使用前请先复制 IDE 中的错误信息到剪贴板
  • 支持语法错误、类型错误、未定义变量等多种错误类型
  • 自动识别光标位置对应的错误信息
